Before running the application, please make sure to install all required dependencies and download necessary files.

Prerequisites

Python installed on your system (recommended: Python 3.10+).

Required files:

requirements.txt

info_kb.csv

action_kb.csv

Make sure all these files are in the project directory.

Installation

1.Install the required Python packages by running:
pip install -r requirements.txt

2.Ensure info_kb.csv and action_kb.csv are in the same directory as app.py.

Running the Application

Run the Streamlit app using the following command in your terminal:
py -m streamlit run app.py

This will start the application, and you can access it in your browser at http://localhost:8501.
